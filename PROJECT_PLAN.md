# 프로젝트: 우리 동네 생활 정보 웹사이트

## 목표
공공데이터포털(data.go.kr)에서 지역 행사/축제/지원금 정보를 자동 수집하고,
AI가 매일 블로그 글을 자동 작성하며,
Google AdSense + 쿠팡 파트너스로 수익화하는 웹사이트

## 기술 스택
- **Next.js (App Router)**: 웹사이트의 뼈대를 만드는 최신 도구예요. (TypeScript & Tailwind CSS 포함)
- **Gemini API**: 구글의 AI로 블로그 글을 자동으로 써주는 역할을 해요.
- **공공데이터포털 API**: 정부나 지자체에서 제공하는 동네 정보를 가져오는 창구예요.
- **GitHub Actions**: 매일 정해진 시간에 컴퓨터가 알아서 일을 하게 만드는 자동 비서예요.
- **Cloudflare Pages**: 만든 웹사이트를 전 세계 사람들이 볼 수 있게 무료로 올려두는 공간이에요.

## 페이지 구성
1. **메인 페이지**: 이번 달 행사/축제 + 지원금/혜택 정보를 카드 형태로 한눈에 보여줘요.
2. **상세 페이지**: 특정 행사나 혜택을 누르면 자세한 내용을 보여주는 전용 페이지예요.
3. **블로그 목록**: AI가 쓴 재미있는 지역 소식 글들을 모아둔 곳이에요.
4. **블로그 상세**: 개별 블로그 글을 읽는 페이지예요.

## 수익화
- **Google AdSense**: 웹사이트 곳곳에 구글 광고를 달아서 광고 수익을 얻어요.
- **쿠팡 파트너스**: 추천 상품 배너를 달고, 클릭이나 구매가 발생하면 수수료를 받아요.

## 자동화 (GitHub Actions)
매일 아침 7시(한국시간)에 비서(GitHub Actions)가 다음 일들을 자동으로 처리해요:
1. 최신 동네 정보(행사, 지원금 등)를 수집해요.
2. Gemini AI에게 "이 정보들을 바탕으로 친절한 블로그 글을 써줘!"라고 시켜요.
3. 새로 쓴 글을 웹사이트에 업데이트해요.
4. 업데이트된 내용을 인터넷에 자동으로 발행해요.

## 환경변수 (나중에 .env.local에 저장)
*이 값들은 비밀번호와 같으니 나중에 안전하게 보관할 거예요.*
- `GEMINI_API_KEY`: AI 사용권
- `PUBLIC_DATA_API_KEY`: 공공데이터 사용권
- `NEXT_PUBLIC_ADSENSE_ID`: 구글 광고 ID
- `NEXT_PUBLIC_GA_ID`: 방문자 분석 ID
